## :pencil: 개요

- 실시간 암호화폐 가격변동 사항을 확인할 수 있는 Crypto Tracker 웹앱입니다.
- 순위별 코인을 한 눈에 확인할 수 있습니다.
- 각 코인별 상세 정보를 확인할 수 있습니다.
  - 현재 코인 가격, 시가총액, 등락률, ATH, 총 발행량 등을 확인할 수 있습니다.
  - 최근 한달간의 변동가격을 캔들 차트로 확인할 수 있습니다.
- 다크모드를 사용할 수 있습니다.
